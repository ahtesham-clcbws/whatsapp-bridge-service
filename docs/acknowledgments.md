## 🛠️ Technology Stack
The bridge is powered by a curated selection of elite open-source technologies:

- **[Baileys](https://github.com/WhiskeySockets/Baileys)**: The industry-standard lightweight WhatsApp Web API library.
- **[Express](https://expressjs.com/)**: The minimalist framework for the RESTful API server.
- **[VitePress](https://vitepress.dev/)**: The high-performance documentation engine powering this portal.
- **[SQLite](https://sqlite.org/)**: Reliable, zero-config local persistence for audit logs.
- **[Pino](https://getpino.io/)**: Low-overhead, professional JSON logging.
- **[Multer](https://github.com/expressjs/multer)**: Robust middleware for handling multipart/form-data media uploads.
