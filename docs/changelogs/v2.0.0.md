# Changelog v2.0.0 - Group Support & Hardening

**Release Date:** 2026-04-10
**Summary:** This major update evolves the WhatsApp Bridge into a group-aware messaging platform with improved stability and security.

### 🚀 New Features
- **Group discovery**: Added `GET /session/groups` to list all participating groups.
- **JID-Agnostic Dispatch**: The `/send` endpoint now natively supports both phone numbers and Group JIDs (`@g.us`).
- **Resilient JSON Parsing**: Improved API robustness to handle standard JSON dispatches even when the `Content-Type` header is missing.

### 🛡️ Security & Sanitization
- **Privacy First**: Completely sanitized all documentation (README and API guides) to remove real phone numbers and developer credentials.
- **Log Security**: Added `X-API-KEY` header support for all administrative endpoints.

### 🏗️ Architecture
- **Version Bump**: Major jump to v2.0.0 representing the core structural shift to group support.
