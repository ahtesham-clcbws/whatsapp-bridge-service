# v3.8.5 (Heavy-Lift Optimization)

**Release Date:** 2026-04-17

This critical stability patch implements "Heavy-Lift" optimizations designed to handle the encryption and network processing constraints of legacy hardware (e.g., MacBook Air 2014) and slow network environments.

---

## ⚡ Legacy Hardware Hardening
- **Extreme Timeouts**: Increased `connectTimeoutMs` and `queryTimeoutMs` to **120 seconds**, preventing "Timed Out" errors during complex encryption handshakes.
- **Cool-down Retry Engine**: Implemented a **5-to-8 second delay** between delivery retries to allow the CPU to recover from background sync tasks.
- **Network Silence on Startup**: Disabled `markOnlineOnConnect` and `fetchOrder` to minimize the initial data burst upon socket establishment.

## 🛡️ Stability Improvements
- **Presence Buffer**: Increased the Presence Warm-up buffer to ensure iPhone recipients always have enough time for an encryption handshake before message delivery.
- **Retry Logic Serialization**: Improved the `sendBatch` loop to ensure that database audit logs are updated even during aggressive hardware-level failures.

## 🎨 UI & Polish
- **CSS Compliance**: Corrected invalid `ring-color` properties in the login screen for better browser standard compliance.
- **Header Synchronization**: Locked all dashboard versions to 3.8.5.
