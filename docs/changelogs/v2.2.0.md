# Changelog v2.2.0 - Webhook Engine & Delivery Tracking

**Release Date:** 2026-04-10
**Summary:** Milestone update introducing outbound connectivity and real-time delivery lifecycle tracking.

### 🚀 New Features
- **Webhook Notification System**: Dispatches real-time POST notifications to a configured `WEBHOOK_URL` for incoming messages and delivery status updates.
- **Delivery Lifecycle Tracking**: The service now tracks and records message status from `Sent` to `Delivered` and finally `Read`.
- **Axios Integration**: Added `axios` dependency for high-reliability HTTP notifications.

### 🏗️ Database & Migration
- **Auto-Migration Engine**: Real-time database schema upgrades.
- **Core Tracking Columns**: Added `whatsapp_id` and `retry_count` to the `audit_logs` table for all current and future weekly logs.

### 🏗️ Architecture
- **Version Bump**: Increment to v2.2.0.
