# Changelog v1.2.0 - Audit Logging

**Release Date:** [Historical]
**Summary:** Introduction of long-term persistence and audit capabilities.

### 🚀 New Features
- **SQLite Audit Engine**: All message dispatches are now recorded in a local SQLite database.
- **Historical Visibility**: Tracks recipient, status (Scheduled/Sent/Failed), message type, and error metadata.
- **Disk Efficiency**: Using SQLite for low-overhead logging suitable for small VPS instances.
