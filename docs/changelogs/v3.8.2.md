# v3.8.2 (Unified Console Insight)

**Release Date:** 2026-04-17

This release brings PM2-style diagnostic visibility directly into the Admin Dashboard, bridging the gap between raw process output and formatted system logs.

---

## 📺 Unified Console Feature
- **Consolidated Stream**: Introduced a new `bridge-console-YYYY-MM-DD.log` file that captures all stdout and stderr output in a single place.
- **Dashboard Integration**: Added a "CONSOLE (ALL)" view to the Log Monitor, allowing administrators to see the raw, unfiltered bridge sequence.
- **Fail-Safe Rendering**: Implemented a "Raw Mode" in the dashboard UI that handles non-JSON strings gracefully, ensuring that library-level console prints (like Baileys) are always visible.

## 🛠 Stability Tweaks
- **Log Buffering**: Optimized the log write cycle to reduce I/O wait times during high-volume events.
- **Filter Pre-sets**: Validated that level-specific filters still function correctly alongside the new console view.
