### 🚀 New Features
- **Headless WhatsApp Bridge**: Bare-bones service based on Baileys to connect a WhatsApp account via QR code.
- **RESTful API**: Basic HTTP endpoints to interact with the linked device.
- **Environment Driven**: Configurable via simple shell variables.
